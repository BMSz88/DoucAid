About DocuAid

DocuAid is an AI-powered chatbot that helps users find information within documents, articles, and webpages. It provides intelligent answers to questions about the content you're viewing.

Key features include:

  • Content extraction from webpages
  • Natural language understanding
  • Context-aware responses
  • Document navigation assistance
  • User authentication and chat history

How to Use

The DocuAid chatbot appears in the bottom right corner of the screen. Click on the chat icon to open the interface. You can ask questions about the content on this page, and DocuAid will provide relevant answers.

Example questions to try:

  • "What is DocuAid?"
  • "What are the main features of DocuAid?"
  • "How do I use this chatbot?"
  • "Tell me about user authentication"

Authentication Features

DocuAid now includes user authentication with the following capabilities:

  • User registration with username, email, and password
  • Secure login with JWT authentication
  • Password hashing for security
  • Persistent chat history across sessions
  • Profile management

Your chat history is stored securely and can be accessed when you log in from any device.

Technical Information

DocuAid is built with a robust tech stack:

  • Frontend: HTML, CSS, JavaScript
  • Backend: Node.js with Express
  • Database: MongoDB for user data and chat history
  • Authentication: JWT (JSON Web Tokens)
  • AI: OpenAI GPT models
  • Vector Database: Pinecone for content indexing

The system architecture follows a client-server model with RESTful API endpoints for communication.
